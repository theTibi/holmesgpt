    Built-in Toolsets¶

    Kubernetes & Containers¶

    Core Kubernetes access — pick one read-only toolset, plus the remediation MCP if you need write actions:

    • Kubernetes — default, read-only via kubectl
    • Kubernetes (MCP) — read-only with OAuth/OIDC
    • Kubernetes Remediation (MCP) — write actions (restart, scale, drain)
